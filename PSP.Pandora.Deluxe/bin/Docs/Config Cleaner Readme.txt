Config Cleaner 3.00 by Smokescreen [4.XX Kernel]
For SLIM and FAT.

___________________________________________________________________________________________
___________________________________________________________________________________________


How to use:
Start Config Cleaner and select a option depending on your needs.

Install:
Copy PSP Folder to your PSP [set GAME folder to 3.XX/ 4.XX]

Operations:

CLEAN CONFIG.SE:
Delete config.se in flash1

BACKUP REGISTRY:
Backup registry files from flash1 to ms0:/CC_BACKUP (Networksettings, Username....)

RESTORE REGISTRY:
Resore registry files from ms0:/CC_BACKUP

DUMP NANDIMAGE:
Dump the internal nand > ms0:/nand-dump.bin
Needs (33MB FAT / 66MB Slim) free space on ms0:

FORMAT FLASH1:
Delete all known files in flash1


Select = PSP model ID

Notes: path is hardcoded to psp/game/configcleaner300

GERMAN TRANSLATION


Config Cleaner 3.00 by Smokescreen [4.XX Kernel]

Für SLIM und FAT.

Installieren:
Kopiere den PSP-Ordner auf die PSP [GAME Ordner im Recovery Menü auf 3.xx/ 4.xx Kernel stellen]


Nutzung:

CLEAN CONFIG.SE:
löscht die config.se im Flash1

BACKUP REGISTRY:
Erstellt ein Backup der Registry Files in ms0:/CC_BACKUP (Netzwerkeinstellungen, Benutzername...)

RESTORE REGISTRY:
Wiederherstellen der Registry Files von ms0:/CC_BACKUP.

Dump Nand-Image:
Erstellt ein Image vom Nand > ms0:/nand-dump.bin
Benötigt (33MB FAT / 66MB Slim) freien Speicher auf ms0:

FORMAT FLASH1:
Löscht alle bekannten Dateien in flash1.


Select = PSP model ID

Notes: path is hardcoded to psp/game/configcleaner300
